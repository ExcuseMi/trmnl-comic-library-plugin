# trmnl-more-comics-plugin

A TRMNL plugin that aggregates hundreds of comic RSS feeds to provide daily comics on your display.

## Features

- Access hundreds of comics from [ComicCaster](https://comiccaster.xyz)
- Support for regular comics, political cartoons, and multiple languages
- Show the latest comic or a random one from your selection
- Add your own custom RSS feeds
- Automatic daily validation of all feeds
- Image filters to optimize display on your TRMNL device

## How It Works
